## 第 3 题

**题目：** 客户端和服务器 TCP 三次握手和传递数据过程如下图所示，请给出图中问号所代表的值。

图中已知：客户端第一次发送 `SYN=1, Seq=100`；服务器应答中有 `SYN=1, ACK=1, Seq=200, ack=?`；随后客户端确认；之后客户端发送 `100Bytes` 数据，服务器发送 `200Bytes` 数据，最后客户端确认。

**答案：**

| 序号 | 方向 | 报文内容 |

| 1 | Client → Server | `SYN=1, seq=100` |
| 2 | Server → Client | `SYN=1, ACK=1, seq=200, ack=101` |

| 5 | Server → Client，发送 200Bytes | `ACK=1, seq=201, ack=201` |
| 6 | Client → Server | `ACK=1, seq=201, ack=401` |

## 第 4 题

**题目：** TCP 的拥塞控制有哪几种算法？如果网络分别出现超时和收到对一个报文段的三次重复确认，那么它们应该分别采取什么算法？门限值 `ssthresh` 如何调整？

**答案：**

TCP 拥塞控制算法主要有：**慢开始、拥塞避免、快重传、快恢复**。

- 出现超时：采用 **慢开始**。`ssthresh = 当前 cwnd / 2`，通常不小于 `2 MSS`；`cwnd` 置为 `1 MSS`。
- 收到三个重复确认：采用 **快重传 + 快恢复**。`ssthresh = 当前 cwnd / 2`，通常不小于 `2 MSS`；重传丢失报文段，之后进入快恢复/拥塞避免阶段。

## 第 5 题

**题目：** 常见计算机网络设备有哪些？交换机和路由器区别？

**答案：**

常见网络设备有：**网卡、中继器、集线器、网桥、交换机、路由器、网关、无线 AP、防火墙、调制解调器**等。

| 对比项 | 交换机 | 路由器 |
|---|---|---|
| 主要工作层次 | 数据链路层 | 网络层 |
| 主要依据 | MAC 地址 | IP 地址和路由表 |
| 主要作用 | 连接同一局域网内设备，转发帧 | 连接不同网络/不同子网，转发分组 |
| 典型功能 | 学习 MAC 地址、交换以太网帧 | 路由选择、隔离广播域、连接外网 |

## 第 7 题

**题目：** 一网络的现在掩码为 `255.255.255.248`，问该网络能够连接多少个主机？

**答案：**

`255.255.255.248` 对应 `/29`，地址总数为 8 个，可用主机数为 `8 - 2 = 6`。

**该网络能够连接 6 个主机。**

## 第 11 题

**题目：** 简述计算机网络层间体系结构的工作原理？OSI 七层协议模型分哪几层？

**答案：**

计算机网络采用分层体系结构：每一层使用下一层提供的服务，并向上一层提供服务；不同主机的同层之间按照相应协议进行逻辑通信。发送方从高层到低层逐层封装，接收方从低层到高层逐层解封装。

OSI 七层模型从低到高为：**物理层、数据链路层、网络层、运输层、会话层、表示层、应用层**。

## 第 13 题

**题目：** 请手写 TCP 三次握手和四次挥手的过程。

**答案：**

TCP 三次握手：

| 次数 | 方向 | 报文 |
|---:|---|---|
| 第一次 | 客户端 → 服务器 | `SYN=1, seq=x` |
| 第二次 | 服务器 → 客户端 | `SYN=1, ACK=1, seq=y, ack=x+1` |
| 第三次 | 客户端 → 服务器 | `ACK=1, seq=x+1, ack=y+1` |

TCP 四次挥手：

| 次数 | 方向 | 报文 |
|---:|---|---|
| 第一次 | 主动关闭方 → 被动关闭方 | `FIN=1, seq=u` |
| 第二次 | 被动关闭方 → 主动关闭方 | `ACK=1, ack=u+1` |
| 第三次 | 被动关闭方 → 主动关闭方 | `FIN=1, seq=v` |
| 第四次 | 主动关闭方 → 被动关闭方 | `ACK=1, ack=v+1` |
